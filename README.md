# Frontend

Nuxt 3 frontend for AI Music Chord Analyzer.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Features

- Modern, responsive UI with subtle blue color scheme
- Drag-and-drop file upload
- Real-time analysis progress
- Interactive chord playback using Tone.js
- AI-powered music theory explanations
- Chat interface for asking questions about songs

## Tech Stack

- Nuxt 3
- Tone.js (audio synthesis)
- Tonal.js (music theory)
- Axios (API calls)
